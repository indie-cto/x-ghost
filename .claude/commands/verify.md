# Verify Changes

Run all verification steps to ensure code quality.

## Steps

1. Run linting: `npm run lint` (or equivalent)
2. Run type checking: `npm run typecheck` (or equivalent)
3. Run tests: `npm test` (or equivalent)
4. Run build: `npm run build` (or equivalent)

## On Failure

If any step fails:
1. Analyze the error
2. Fix the issue
3. Re-run all verification steps

Report the final status of all checks.
